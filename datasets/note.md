1. [Thai Toxicity Tweet Corpus](https://github.com/tmu-nlp/ThaiToxicityTweetCorpus):
Thai Toxicity Tweet Corpus is a dataset of Thai tweets collected using 44 toxicity-related keywords and manually annotated for toxic content. It contains around 3,300 tweets (with some missing text), making it suitable for training toxicity detection models in Thai social media. The corpus is hosted on GitHub by tmu-nlp and provides guidelines for human annotation.

2. [SEACrowd – Thai Toxicity Tweet Dataset](https://huggingface.co/datasets/SEACrowd/thai_toxicity_tweet):
SEACrowd – Thai Toxicity Tweet Dataset is a hosted version of the Thai Toxicity Tweet Corpus on Hugging Face, curated as part of the SEACrowd initiative for Southeast Asian languages. It retains the original 3,300 tweets focused on toxicity classification, enabling easy loading for NLP tasks like content moderation.

3. [Wisesight Sentiment Dataset (Thai)](https://github.com/PyThaiNLP/wisesight-sentiment):
Wisesight Sentiment Dataset (Thai) is a large collection of 26,737 informal Thai social media texts (posts, headlines, ads) labeled for sentiment analysis across categories like neutral, positive, and negative. Released by PyThaiNLP, it includes train/test splits, tokenization samples, and is under CC0 license for Thai text classification benchmarks.

4. [HateThaiSent Dataset](https://github.com/dsmlr/HateThaiSent):
This dataset includes 7,597 Thai social media messages annotated for both hate speech (Hatespeech/Nonhatespeech) and sentiment (Positive/Neutral/Negative), supporting multi-task models for improved hate detection.

5. [Jigsaw Toxic Comment Classification Challenge](https://huggingface.co/datasets/thesofakillers/jigsaw-toxic-comment-classification-challenge):
This Kaggle dataset from Wikipedia comments contains ~160k training samples labeled across six toxicity types: toxic, severe_toxic, obscene, threat, insult, identity_hate (multi-label).

6. [Toxic Tweets Dataset](https://www.kaggle.com/datasets/ashwiniyer176/toxic-tweets-dataset):
A balanced Kaggle collection of ~15k tweets labeled binary as toxic/non-toxic, focused on hate speech and offensive language in Twitter data.

7. [Hate Speech and Offensive Language Dataset](https://www.kaggle.com/datasets/mrmorj/hate-speech-and-offensive-language-dataset):
From Kaggle, this ~25k Twitter dataset labels tweets as hate_speech, offensive_language, or neither, aiding fine-grained classification.

8. [Hate Speech Dataset for Social Media](https://www.kaggle.com/datasets/ziya07/hate-speech-detection-dataset-for-social-media): 
1.8k synthetic Twitter/Reddit posts labeled neutral/offensive/hateful.
